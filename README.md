# Manual QA Testing Project

## Overview
This repository contains a manual QA testing project created to demonstrate software quality assurance skills.

## Project Objectives
- Apply manual testing methodology
- Create a test plan
- Design functional test cases
- Document software defects

## Test Scope
- User login functionality
- Field validation
- Error handling

## Test Types
- Functional testing
- Validation testing
- Exploratory testing
- Smoke testing

## Tools Used
- GitHub
- Markdown

## Repository Structure
- test-cases: manual test cases
- bug-reports: documented defects
- test-plan: test strategy and scope
- test-scenarios: high-level scenarios (Gherkin)
- test-data: datasets used during testing
- checklists: smoke/regression checklists
- templates: reusable QA templates
- evidence: screenshots/videos/logs
